Window:

Window is the first thing that gets loaded into the browser.It is global object.window is the root of everythingyou can call window.screen and window.document to get those elements

It has some properties like:

  1. length
  2. innerWidth
  3. innerHeight
  4. name
  5. open
  6. close

Document

Each window object has a document object to be rendered. These objects get confused in part because HTML elements are added to the global object when assigned a unique id. E.g., in the HTML snippet

It has some properties like:

  1. url
  2. cookies

Screen:

The window object also has a screen object with properties describing the physical display.The portion of a screen displaying the rendered document is the viewport in JavaScript, which is potentially confusing because we call an application's portion of the screen a window when talking about interactions with the operating system.

It has some properties like:

  1. width
  2. height
